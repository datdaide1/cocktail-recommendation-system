# Data Pipeline

This folder contains scripts to process raw data crawled from the internet, exist in "E:\RECOMMENDATION-SYSTEM\crawled-data", generate embeddings, and insert them into Qdrant Vector DB.

## How to use

1. Install dependencies:
   ```bash
   conda activate cocktail-ai
   cd data-pipeline
   pip install -r requirements.txt
   ```
2. Run pipeline scripts (to be implemented).
